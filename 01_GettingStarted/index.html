What is TypeScript?

  • A JavaScript Superset
  • A Language building up on JavaScript
  • Adds new Features + Advantages to JavaScript
  • Browser CAN'T execute it!
  • Features are compiled to JS "workarounds", possible errors are thrown

Why TypeScript?

    It prevent unwanted behaviour at runtime
  • Validate & sanitize user input

    • Installing and using

    In order to have typescript it is necessary to have it installed globally

    command to compile ts code into js: tsc |filename|.ts

    TypeScript overview

    • Typescript adds types
    • Applies next-gen JavaScript features (compiled down with old browsers)
    • Uses non JavaScript features (like Interface or Generic)
    • Meta-Programming features like Decorators
    • Rich configuration options
    • Modern tooling that helps even in non-Typescript projects
